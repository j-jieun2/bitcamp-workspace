- 채팅 등 다른 테이블에서 사용할 하이워크를 사용하는 사람의 기본 정보테이블
- **이메일**
- 암호
- 사진
- 부서번호
- 권한번호
- 사원번호
- 캘린더번호
- 채팅방번호
- 게시판번호


* 권한번호
  - 권한명 
  예)  0. 사원 1. 인사팀 2.관리자 등 사용자에 따라 권한을 부여한다.


* 사원 정보 : 사원의 세부 정보를 담는 테이블
  - **사원 번호**
  - 연락처
  - 성별
  - 연차
  - 입사일
  - 재직상태


* 부서 정보 : 사원의 부서와 직급정보를 담는 테이블
  - **부서 번호**
  - 부서명
  - 직급
